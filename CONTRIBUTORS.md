## How to Get Listed

Contributors are automatically recognized when they:

1. **Submit a Pull Request** that gets merged
2. **Report a Bug** that helps improve the project
3. **Contribute to Discussions** with valuable insights
4. **Help Others** in issues or community channels
5. **Improve Documentation** with corrections or additions
6. **Create Content** like tutorials, blog posts, or videos

### Manual Recognition

If you've contributed but aren't listed, please:
1. Open an issue with the title "Add Contributor: [Your Name]"
2. Describe your contribution
3. A maintainer will add you to the list

---

## Becoming a Maintainer

Consistent contributors may be invited to become maintainers. Maintainers have:

- Write access to the repository
- Ability to merge pull requests
- Voice in project direction decisions
- Recognition on the project homepage

### Maintainer Responsibilities

- Review and merge pull requests
- Triage and respond to issues
- Help maintain code quality
- Support and mentor new contributors
- Uphold the Code of Conduct
